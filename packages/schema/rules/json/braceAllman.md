**Default** `false`

💁🏽‍♀️ &nbsp;&nbsp; Recommended setting is: `true`

The `braceAllman` rule puts JSON braces onto new lines, producing an [Allman Style](https://en.wikipedia.org/wiki/Indentation_style#Allman_style) output.

---

#### 👎 &nbsp;&nbsp; Disabled

```json
[
  { "prop": "value" },
  { "prop": "value" },
  { "prop": "value" }
]
```

---

#### 👍 &nbsp;&nbsp; Enabled

```json
[
  {
    "prop": "value"
  },
  {
    "prop": "value"
  },
  {
    "prop": "value"
  }
]
```
